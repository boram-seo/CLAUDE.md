# Project Rules

## 디자인 시스템
- Figma 디자인 작업 시 반드시 `design.md`의 디자인 시스템을 따를 것
- 색상, 타이포그래피, Border Radius, 그림자 등 모든 토큰은 design.md 기준
- Primary 색상(`#2943f9`)은 가장 중요한 CTA 버튼 및 핵심 정보에만 사용
- Accent 색상(`#ff6f43`)은 전체 UI 중 5% 이하로 제한
- Border Radius는 최대 12px (완전한 원형 제외)
- 폰트는 Pretendard (Regular 400, SemiBold 600, Bold 700)

## Figma 스킬
- `.claude/skills/` 폴더의 Figma 스킬들을 활용하여 디자인 작업 수행
- 스킬 원본: https://github.com/figma/mcp-server-guide
